# Kodluyoruz Ilk Repo

## Installation
---------------------------------------------------------------
Öncelikle projeyi clonelayın. 
```
git clone https://github.com/alimertkarakus/kodluyoruzilkrepo.git
```
## Usage
-----------------------------------------------------------
Projeyi cloneladıktan sonra Visual Studio Code programını açınız.
Linux için:
```
cd kodluyoruzilkrepo
cd.
```
